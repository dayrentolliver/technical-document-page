Front-End Cheat Sheet

*Disclaimer - This material was heavily inspired by freeCodeCamp.*
Introduction

This is a super quick guide is designed to give a high-level overview of the skills needed to become a front-end developer. I've also provided a listed material that I've found helpful, or a lot of others have found useful. This guide is not intended to be all encompassing. At the very least, it should hopefully pique your curiosity about becoming a programmer.

HTML

Hyper Text Markup Language, or HTML, as it is most commonly referred to, is the most basic part of any webpage. Here are a few example HTML tags below:

<!DOCTYPE html>
<html>
<head>
<title></title>
</head>
<body>
<h1></h1>
<p></p>
</body>
</html>

Resources to learn HTML:

  • MDN Beginner HTML tutorials
  • A YouTube video by Traversy Media
  • freeCodeCamp's hands on material
CSS

Cascading Style Sheets, or CSS, has the ability to change the way your entire website's layout, from different colors to fonts. Below is an example of CSS:

h1 {
color:red;
text-align: center;
}

Resources to learn CSS:

  • freeCodeCamp's Basic CSS course
  • Can't go wrong with another video by Traversy Media
  • CSS Tricks also has handy content
JavaScript

JavaScript has the ability to make your website more lively. In fact, it's used on more than 90% of sites on the web! Here's an example of a JavaScript function below:

function add(a,b) {
return a + b;
}

Resources to learn JavaScript:

  • Learn JavaScript from the MDN's beginner tutorials
  • Go more in depth with a Udemy course
  • Codecademy offers a great course to help you JavaScript
ES6

What's the difference between ES6 and JavaScript, you ask? Well since the introduction of ES6 in 2015, some key things, like how to declare a variable have changed.

let lastName = 'Smith';
const birthMonth = 'May';

Resources to learn ES6:

  • Read Eloquent JavaScript for free here
  • Watch this YouTube tutorial
